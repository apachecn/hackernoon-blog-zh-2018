编码波束SF —加利福尼亚州旧金山

今年3月，光束以编码光束SF在旧金山着陆。还记得二郎&仙丹厂吗？2018年，我们重新思考了一些事情，并将其重新命名为Code BEAM SF。点击了解更多关于过渡的信息。前两个主题演讲 Kostis Sagonas 和 Richard Cook 已经确定！

您需要了解的内容
地点:旧金山海军陆战队纪念俱乐部&酒店
日期:2018年3月15日星期四—16日星期五
售票活动: 早鸟票已发售

Lonestar ElixirConf 2018德克萨斯州奥斯汀

今年二月，长生不老药带着Lonestar ElixirConf 来到德克萨斯州。第二年，《孤独之星》的主题演讲来自凤凰框架炼金术士克里斯·麦考德、神经核心团队成员蒂姆·梅克莱姆和亚伦·帕特森、鲁比核心团队成员。

你需要知道的
地点:德克萨斯州奥斯汀诺里斯会议中心
日期:2018年2月22日星期四—23日星期五
售票活动: 标准票有效期至2月3日

2018年洛杉矶展——加利福尼亚州洛杉矶

EMPEX是在洛杉矶举行的关于Elixir编程语言和生态系统的复杂会议系列。今年，EMPEX将在洛杉矶市中心的一个娱乐场所举办一场技术讲座，包括艾玛·坎宁安和T2·莎拉·格雷的主题演讲。

你需要知道的事情
地点:加州洛杉矶南希望街1827号90015
时间&日期:太平洋标准时间2018年2月10日星期六
购票活动:获取门票此处

Lambda Days —波兰克拉科夫

Lambda Days是函数式编程世界中独一无二的体验。主题演讲 Mary Sheeran 和 Philip Wadler 将与 Heather Miller 和 José Valim 一起参加今年二月的Lambda Days，为期两天，共分六场。

你需要知道的事情
地点:礼堂最大UJ、克拉科夫
日期:2018年2月22日星期四—2月23日星期五
售票活动: 普通票仍有

ElixirDaze 2018 —科罗拉多州丹佛市

ElixirDaze是一个建立Elixir社区的会议，提供顶级演讲和有用的黑客技术。根据新闻报道，今年的主题演讲将会是罗布·科内瑞和弗雷德·赫伯特

Code BEAM Lite —意大利米兰

今年4月，Code BEAM Lite在米兰举办了Code Sync 与 Coders51 的见面会。呼吁谈判仍然开放！本次活动将围绕工具、BEAM、框架、分发、并发和多核等主题，由 Francesco Cesarini 、 Gabriele Santomaggio 、 Gianluca Randazzo 和 Paolo Laurenti 组成的恒星计划委员会策划，带来许多精彩的演讲想法。已经确认的有Erlang的共同发明人Mike Williams 和软件顾问Georgina McFadyen 。

你需要知道的事情
地点:维尼尼42，Via Giulio e Corrado Venini，米兰
日期:2018年4月6日星期五
已出票活动: 早鸟票可得

ElixirConf欧盟—波兰华沙

继去年在巴塞罗那举办的ElixirConf EU 2017大获成功之后， ElixirConf EU 今年又重返欧洲大陆，这次是在华沙！ElixirConf EU的主题演讲人包括José Valim、Chris McCord和Wu。还提供培训。

日期:2018年4月16日星期一—18日星期三
已出票会议: 早鸟票仍有
