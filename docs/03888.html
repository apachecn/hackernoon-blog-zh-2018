  1. 第一个想法是，所有的技术变革都是一种交换……[因此]新技术提供的每一个优势，总会有相应的劣势……[而且]一项技术的奇迹越大，它的负面后果就越大。想想汽车，尽管它有很多优点，却污染了我们的空气，堵塞了我们的城市，并破坏了我们的自然景观。印刷术给了我们……国家地位，但这样做却把爱国主义变成了肮脏的[情感】…。
  2. 【第二】新技术的优势和劣势从来不是在人群中平均分配的……[计算机]对大型组织有利……[但是]在多大程度上[它]……对大众有利……[他们]更容易被跟踪和控制……对有关他们的决策越来越迷惑……[以及]广告公司和政治机构的容易目标……胜利者总是说……我们掌握的信息越多越好。但这有多真实呢？如果世界上有儿童在挨饿——确实有——这并不是因为信息不足。

  5. 【第五】媒体倾向于成为神话……【好像】自然秩序的一部分……而不是在特定的政治和历史背景下产生的人工制品。当一项技术变得神秘时，它是…危险的，因为它被接受了，而且…不容易被修改或控制…[作为]一种偶像崇拜的形式…[所以]对它的益处的信仰可能是一种错误的绝对…它的善恶能力完全取决于人类对它为我们和我们自己做了什么的认识。…

  4. 帮助人们管理身体和财务风险。
