VueJS 是“渐进式 JavaScript 框架”。它从视图库和前端框架世界的所有现有技术中获得灵感，包括AngularJS、React、Angular、Ember、Knockout和Polymer。在Vue(和Angular/AngularJS)中，指令是一种包装通常应用于DOM元素的功能的方式。Vue文档中的例子是一个focus指令。
