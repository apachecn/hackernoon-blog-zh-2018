基于组件的软件工程(【CBSE】)，也称为基于组件的开发 ( CBD )，是软件工程的一个分支，它强调对给定软件系统中广泛可用的功能进行关注点分离。

在基于组件的架构高度流行的今天，每个现代UI框架都提供了组件结构的方法。看看这篇文章的利弊。我在CBA最喜欢的就是关注点的分离😎

无论是开始一个小的项目还是有一个大的项目，关注点的分离都将是易维护性、可测试性、可重用性的关键点。如果你是UI工程师，你应该听说过 npm 包管理器。这是将可重用组件和服务转移到那里的好方法。

4。测试
5。语义版本
6。私有或公共(社区可能有助于改善)
