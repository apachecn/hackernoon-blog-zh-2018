NumPy是使用Python进行科学计算的基础包。

在Python数据科学系列的第1部分中，我们研究了Python中数值计算的基本内置函数。在这一部分，我们将看看Numpy库。

NumPy是使用Python进行科学计算的基础包。除其他外，它包含:

  • 一个强大的N维数组对象
  • 复杂的(广播)功能
  • 集成C/C++和Fortran代码的工具
  • 有用的线性代数、傅立叶变换和随机数功能

除了其明显的科学用途，NumPy还可以用作通用数据的高效多维容器。可以定义任意的数据类型。这使得NumPy可以无缝、快速地与各种数据库集成。

很好，让我们看看如何使用Numpy库进行基本的数组操作。

Numpy图书馆

import numpy as np
