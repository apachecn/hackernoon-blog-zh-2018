**1。澳大利亚将在检查 ico 时实施金融市场加密规则**

主要金融监管机构澳大利亚证券和投资委员会(ASIC)发布了 2018-2022 年的[企业计划。这项倡议是跨部门的努力，将采取监督的办法。文件中提到](https://download.asic.gov.au/media/4855947/asic-corporate-plan-2018-22-focus-2018-19-published-31-august-2018.pdf)[将密切关注新兴技术，包括 ico 和加密货币](https://www.coindesk.com/australian-watchdog-to-apply-market-rules-to-crypto-exchanges/)。此外，在头两年，ASIC 将实施一个基于金融市场密码交易所规则的监管框架。必要时，它还会进行干预，以确保消费者和投资者得到保护。最后，市场参与者将通过一个[许可系统](https://asic.gov.au/for-finance-professionals/market-infrastructure-licensees/)进行监控。尽管 ASIC 没有为加密交易制定有针对性的法规，但它去年还是为 ICOs 发行者发布了一套指导原则。

**2。中国最高法院允许在法庭上使用区块链认证的证据**

![](img/1814e9d91a0a1b19a6903a594322826e.png)

中国最高法院发布了一份公告，显示通过区块链认证的 T2 证据现在具有法律约束力，可以在法庭上提出。这一决定是一系列规则的一部分，旨在为互联网法院的诉讼程序提供更多清晰度，所有阶段都在网上进行。如果一方能够证明其可靠性，通过区块链收集和存储的数字数据是可以接受的。这是一个最新的例子，表明这些新兴技术有可能彻底改变法律体系。7 月，我们[报道了](/@BlockEx/20-07-2018-biggest-stories-in-the-cryptosphere-219d98c05188)英国法律委员会的[2017/2018 年度](https://assets.publishing.service.gov.uk/government/uploads/system/uploads/attachment_data/file/727386/6.4475_LC_Annual_Report_Accounts_201718_WEB.PDF)报告。在该文档中，讨论了智能合约及其获得的流行度。该独立机构目前正在调查这项技术如何帮助其法院系统保持竞争力。

**3。高盛首席财务官否认他们取消了加密交易部门**

![](img/00e465eed92fe25f00fd3b35d9372ad4.png)

周三，我们报道了 Next Web 的一篇文章，声称高盛无限期推迟了其加密交易平台项目，同时保留了托管服务。然而，[首席财务官(CFO) Martin Chavez 现在否认了这一说法](https://www.cnbc.com/2018/09/06/goldman-sachs-cfo-calls-reports-of-shutting-down-crypto-desk-fake-news.html)，称金融服务公司仍在努力解决这一问题。相反，查韦斯说，由于客户的需求，他们现在正在开发一种“不可交割远期”的比特币衍生品。首席财务官拒绝了这份报告，但也指出时间表尚未确定。他接着说，当他们宣布的时候，他们的意思是这是一个将随着时间的推移而发展的探索，而不是一个有固定日期的项目。目前，高盛正在清算 CBOE 和芝加哥商业交易所发行的比特币期货，同时向其客户提供流动性。

**4。乌兹别克斯坦希望吸引具有加密优势的加密交易所**

![](img/a25ff5e590add3f4658d772032d2d590.png)

乌兹别克斯坦总统沙夫卡特·米尔济约耶夫希望用税收优惠吸引秘密交易。本周早些时候，他向[发布了一项命令](https://static.norma.uz/official_texts/%D0%9E%20%D0%BC%D0%B5%D1%80%D0%B0%D1%85%20%D0%BF%D0%BE%20%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D0%B8%20%D0%B4%D0%B5%D1%8F%D1%82%D0%B5%D0%BB%D1%8C%D0%BD%D0%BE%D1%81%D1%82%D0%B8%20%D0%BA%D1%80%D0%B8%D0%BF%D1%82%D0%BE-%D0%B1%D0%B8%D1%80%D0%B6%20%D0%B2%20%D0%A0%D0%B5%D1%81%D0%BF%D1%83%D0%B1%D0%BB%D0%B8%D0%BA%D0%B5%20%D0%A3%D0%B7%D0%B1%D0%B5%D0%BA%D0%B8%D1%81%D1%82%D0%B0%D0%BD.pdf)，列出了密码交易所迁移到该国将会受益的一系列好处。税收将不适用于与加密相关的收入。此外，处理外国法定货币的特许加密交易所将不受现行外汇条例的约束。他们还将免受证券交易法规的约束。一旦 crypto exchange 在该国开设子公司，就将颁发许可证。但是，还必须提到的是，它们需要满足某些要求。授权资本至少为平均最低工资的 3 万英镑，即 70 万美元。员工必须位于乌兹别克斯坦。加密交易所还需要实施反洗钱规则，并将数据存储至少五年。最后，矿工也会享受一定的福利。那些用电量超过 100 千瓦时的企业将无需拍卖就能获得土地。
