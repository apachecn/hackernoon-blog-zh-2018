更改git提交的日期

原文：https://medium.com/hackernoon/change-the-date-of-a-git-commit-eeed8d2c5b9b?source=collection_archive---------5-----------------------#2018-08-29

使用 git 最大和最糟糕的事情之一就是你可以重写历史。这里有一个卑鄙的滥用方式，我想不出一个合法的理由这样做。

和其他事情一样，感谢StackOverflow 为我提供的所有选择👍。

GIT_COMMITTER_DATE="$(date)" git commit --amend --no-edit --date "$(date)"
