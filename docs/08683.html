市场状况—2018年10月18日
BTC:6540.86美元(+0.12%)
ETH:206.24美元(-0.43%)
XRP:0.46269美元(-0.29)

经过周一的突然飙升和之后的回调，市场保持稳定。事实上，前5大加密货币在过去24小时内的变动不到1%。比特币稳定在6500美元，以太坊继续持有在200美元以上。

其他新闻，总部位于英国的全球安全公司G4S正在开发一种加密货币托管解决方案。该公司通常为活动、监狱和其他更传统的部门提供安全保护，认为它有解决困扰加密货币行业的盗窃风险的解决方案。此外，加密货币合规和调查公司Chainalysis与币安合作打击加密洗钱。Chainalysis将部署其实时加密货币监控程序，该程序使用模式识别、开源参考和复杂算法来跟踪和警告可疑的加密货币交易。

1)基于PoS的智能合约区块链Qtum与亚马逊网络服务(AWS)中国区合作，拓展托管服务的区块链即服务(BaaS)。此次合作将见证在AWS上开发智能合同平台，使用户和开发者能够高效、经济、快速地管理、启动和编码智能合同系统。该解决方案将利用亚马逊机器映像(AMI)以及用于启动、Gmix web IDE和solidity的核心软件。此外，Qtum现在可以访问亚马逊的业务、营销、销售和技术资源。(阅读更多)

在Tether下跌之际，比特币在Bitfinex的交易价格溢价300美元。 Tether (USDT)的市值在短短十天内从28亿美元暴跌至20亿美元以上，交易者兑换比特币导致市值缩水5%。许多人认为，价格上涨是因为通过交易所购买Tether的交易者现在正寻求出售手中的比特币。(阅读更多)

3)芝加哥商品交易所的一份报告显示，2018年第三季度，该交易所的比特币期货交易增长了41%。此外，未平仓合约数量上升了19%。CME还指出，与Q2相比，第三季度的交易量增长明显放缓。彭博最近报道称，芝加哥商业交易所没有推出更多加密货币期货的计划，因为他们更喜欢继续分析比特币期货。BTC期货一直是一个有争议的话题，因为一些人认为它们在BTC泡沫爆炸前帮助稳定了泡沫，另一些人认为该工具对2018年的加密货币熊市负有直接责任。旧金山联邦储备银行也表示，比特币期货推出后“[BTC]价格的后续下跌”似乎不是“巧合”(阅读更多)
