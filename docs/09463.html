BCH SV简介

众所周知，比特币现金(BCH) 经历了一场有争议的硬分叉，产生了两个不同支持的新链 BCH ABC 和 BCH SV。

现在这两种硬币都被开采出来了(具有不同的支持和不同的散列能力)，它们中的每一种都有不同的美元价值，并且现在都在主要的交易所交易。

在这里，我想提供一个详细的指南，告诉你在哪里以及如何购买BCH ABC和BCH SV。

我们开始吧

首先，我们需要使用法定货币购买比特币，以便在其他交易所购买任何加密货币(或者您可以使用您之前购买的比特币)。因此，按照链接在比特币基地购买比特币，然后按照下面的步骤在币安购买 BCH SV 。如果你已经有了比特币，你可以跳过这一步，直接去币安注册。

在币安注册

币安是一家交易费用相对较低的加密货币交易所，到目前为止表现良好，没有任何问题，支持反应良好。币安上的交易平台有两种选择:基础和高级。在这篇文章中，我将主要介绍基本的交易选项，但是你可以查看下面的高级视图。除了能够在高级视图中进行更详细的技术分析之外，这两种视图之间没有太多的区别。

将您的比特币存入币安

一旦你在币安上注册并登录，你可以在右上角看到几个菜单项。其中一个是“基金，有一个子菜单项名为“存款”。在这里你可以看到你所有的存款(从其他地址存入币安)。

因此，为了将我们在比特币基地或双子星购买的比特币发送到币安，我们需要一个比特币地址——我们可以在这个页面上找到。

查看以下截图，了解如何从比特币基地向币安存入比特币

复制地址，把你的BTC从比特币基地 / 双子座寄到这个地址。

一旦你在币安上收到比特币，点击左上角 兑换 下的 基本 子菜单项。

现在我们需要在货币列表中找到带有比特币交易对的 BCH SV 币。看看下面的截图和说明。

  • 选择右上角的BTC盒子。这意味着你在寻找可以用比特币交易的硬币。
  • 在硬币列表上方的搜索框中输入 BCH SV ，它会如下所示过滤你。现在你可以看到 BCH SV 的现价。你几乎接近投资 BCH SV 。现在只需点击 BCH SV 即可。
  • 点击 BCH SV ，左侧显示烛台和订单日线图，如下所示。

点击左侧所需的绿色价格，填写您在BTC的 BCH SV 的价格(在上面的屏幕截图中当前显示为0.022236)。单击后，它会变为0.022251。现在你需要选择你想在 BCH SV 投资多少BTC。点击25|50|75|100%,填写以该价格可以购买的 BCH SV 的金额。

你可以从那些绿色数字中自由选择价格——它们基本上是其他人的不同报价——这也叫限购。

既然你是新手，你也可以点击旁边的 行情 标签，选择BTC金额，点击 买入BCH SV——这基本上是以市场价买入。

一旦显示成功，您可以从右上角菜单进入 资金- >余额 ，搜索 BCH SV。

祝贺您购买了BCH ABC。
