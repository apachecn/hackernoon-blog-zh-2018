从前 …准确的说是2013年12月18日，gamekyubi的 女友在一家女同酒吧。为什么？好吧，嗯，问他。

无论如何，像许多其他书呆子一样，在寒冷的夜晚，他独自一人坐在电脑前，为自己感到难过。我们心爱的比特币正在贬值。当GameKyuubi的女人“和女孩们一起闲逛”的时候，(我不会告诉他她真正在做什么)他发现自己在喝威士忌和网上聊天。#有罪

他感到不太自信，于是登录Bitcointalk.org，发布了一些相当有趣的内容，这些内容直到今天还在继续逗我们开心，或者让我们恼火。他的帖子标题是 “我是霍德林 ”。他承认打错了标题，但在多次尝试纠正错误后，在一种陶醉和情绪化的状态下，他放弃了正确拼写“hold ”,继续完成他的帖子。

显然，没过多久，比特币爱好者们就开始在他的帖子上发表评论、大笑和创意。早期的BTC矿工和商人加入到支持GameK醉醺醺的建议中，并评论说他们也将和他一起去“T10”HODL。就在那时，发明了第一个密码俚语。

为了庆祝“ HODLING ”诞生5周年，我想分解一下其他加密货币俚语和术语的含义。别担心，我还会为加密货币和区块链世界中使用的交易和技术术语提供可靠的参考指南。这里什么都没有？

俚语“术语:有些爱，我不是粉丝#讨厌者

寿星和先锋可以先走。

  • HODL/霍德林:持有/持有

例子:“我到死都在哭！”

  • ATH :“空前高涨”

例子:“去年《生活大爆炸》播出后，BTC看到了ATH。"

  • Bagholder :交易者在一种加密货币大幅贬值后持有该货币。

例子:“你拿着Dogecoin干嘛，你个包包！一文不值！”

  • 看跌:预期价格将会下降

例如:“交易你的NEO！我预测一个看跌的月份！”

  • 看涨:预期价格将会上涨

例子:“我会大量投资XRP，非常确定我们正在进入一个牛市”

  • BTFD:逢低买入！

例子；“我告诉你BTFD混蛋！你为什么不听！?"

  • 热衷于将密码学作为社会变革手段的人。

例如:“雅各布·阿佩尔鲍姆是个巴派朋克！”

  • DYOR:自己做调查

例子:“如果你不DYOR，你就是个白痴！！！!"

  • 翻转:替代币的价值变化，不再主要基于比特币的价值

例子:“Eth比BCH还低！那是轻率的，兄弟！”

  • FOMO :害怕错过

例子:“你们都在做什么，你知道，因为我患有严重的FOMO，我需要知道。”

  • FUD :恐惧、不确定和怀疑(传播这种消极情绪的人被称为“老糊涂”)

例如:“与FUD相关的加密货币总体价格有所下降。”

  • 蓝宝:兰博基尼的简称。

例子:“Alyze说，”很少使用‘兰博’的人实际上是这种极其昂贵的车辆的所有者。"

  • 月:价值大增。

例如:“什么时候月亮？”你会看到“文明月！?"通常，因为当你在赔钱的时候，拼写显然不重要。

  • Nocoiner :不拥有加密货币的人。

例子:“他凭什么认为自己可以给出理财建议？！Nocoiners不能当真，这年头！”

  • Noob :对密码/游戏的侮辱或新人。

例子:“你不擅长交易，新手。”

  • 强迫症:强迫症

例子:“我的家人今年没有邀请我去过感恩节，因为我有强迫症，他们厌倦了我。”

  • 泵&倾卸:价格涨跌循环。

例如:“约翰·麦卡菲付给希尔的任何劣质硬币都是一个抽水和倾倒计划。”

  • Rekt :失事

例如:“我的投资组合是rekt，就像一个mofo！”

  • 【SAFU】:安稳如“傅**。

例如:“今天投资加密，三年后，你将成为SAFU！”

  • 令人讨厌且不必要的广告

例子:“你为什么在我的电报频道上扔你的破硬币？蹩脚。”

  • Sh** Coin :没有内在价值的加密货币。

例句:“ dogecoin 绝对是Sh**币。”

  • 恶搞:当一个人/团队在设定的范围内对自己进行买卖，制造成交量的假象。

例如:“鲸鱼今天在恶搞。”

  • 这是绅士:指出加密货币图表中正面标注的短语。

例子:“今天是个不错的交易日，我的投资组合看起来不错。这就是君子，乡亲们，你们自己看吧！”

  • 蒸汽软件:一种尚未上市的广告技术。想法、计划或概念。

例子:“哥们！听起来棒极了！哪里可以投资！？你说这是蒸汽器皿是什么意思！?"

  • 鲸鱼:拥有大量加密货币的鲸鱼。

例子:“我希望我是一条冷静的鲸鱼，就像蒂姆·德雷珀一样。”

交易术语；《华尔街之狼》对这些秘密事件一无所知……

  • 套利:利用两个或更多市场之间的价格差异。
  • 布林线:描述价格和波动性随时间变化的统计图表。
  • 加密货币“Crypto”:设计用作商品和服务的分散交换的数字资产。
  • CMC :币市值。用于对加密货币的大小进行排名。它的计算方法是将价格乘以流通供应量。
  • 核心:打算长期持有的头寸。
  • DCA :美元成本平均。将期望投资的金额分时段进行，希望降低风险。
  • DEX :去中心化交易所。一个P2P交易所。
  • EMA :指数移动平均线。基于一段时间内价格行为的线。
  • EW :埃利奥特波。资产价格运动趋向形成的模式。
  • 交换；您可以购买、出售或交易加密货币的网站。
  • FA :基本面分析。对影响市场价格的潜在因素的研究。
  • 法定货币:政府认可的货币
  • ICO :首发币发售。加密货币的筹资渠道。
  • IPO :首次公开发行。一家公司的股票出售给投资者。
  • ISO :首次证券发行。使用区块链平台以证券形式进行的预期筹资。
  • 阶梯式:设置增量买入或卖出指令
  • Long :当货币价值增加时，贸易的有利可图的差额。
  • 马:移动平均线。(滚动/运行平均值)计算来分析ta中的数据点，通过使用硬币价格的整体图表平均值找到。形成趋势跟踪指标。
  • MACD :移动平均线收敛/发散。技术分析中使用的交易指标。(见TA)
  • 市值:(市值)流通代币供应量乘以现价。
  • 融资融券:“杠杆交易”。如果你不知道这是什么，就不要做。
  • 市价订单:当前市价的即时或即时订单。
  • 恐慌性抛售:出于恐惧抛售资产。
  • 庞氏骗局:一种投资骗局，涉及从新投资者投入的资金中向现有投资者支付所谓的回报。
  • 风险管理:出售资产的计划。
  • RMC :真钱蜡烛。盈利能力将会增加的一个指标。
  • 路线图:硬币背后的未来详细地图，以及项目背后团队陈述的信心。
  • ROI :投资回报。净利润与投资成本的比率。
  • RSI :相对强弱指标。衡量最近价格变化的幅度。
  • Satoshis (Sats): 比特币的最小分数称为Satoshi。它代表一个比特币的百万分之一。
  • 卖出墙/买入墙:当前买入和卖出订单的图形表示。
  • 空头:当货币价格下跌时的一笔有利可图的贸易差额。
  • 稳定硬币:旨在将价格波动影响降至最低的加密货币。
  • STO :证券代币发行。使公司能够从外部可交易资产中创造象征性价值。
  • 随机振荡器:发现某一资产在某一时间段内的高点和低点之间的范围的指标。
  • 止损单:自动卖单。
  • TA :技术分析。通过研究过去的市场数据来预测价格波动的方法。
  • 获利:在价格上涨后卖出资产。
  • 股票代码:每枚硬币都有交易代码或股票代码。(比特币= BTC，以太= ETH)
  • 代币:驻留在自己的区块链上的加密货币，代表资产或公用事业。

技术术语:服用两片阿司匹林，继续…

  • 2FA:双因素认证。使用两种不同因素的组合来确认用户身份的方法。
  • 空投:免费发放虚拟货币代币。
  • ASIC :专用集成电路。专门的硅芯片，处理SHA-256算法，以“挖掘”加密货币并验证交易。(参见采矿)
  • ASIC矿工:与采矿软件一起使用的机器，用于开采加密货币。(参见采矿)
  • 块:与网络相关的分类帐中不可更改的数据文件被永久保存。永远。(不像你的前两次婚姻。#opps)
  • 区块链:一种数字账本，其中加密交易按时间顺序公开记录。
  • 区块高度:链上genesis区块(第一个开采的区块)之前的区块数。
  • 方块奖励:为解决与方块相关的数学方程式而分配的报酬。(哈希+解决的问题=奖励)
  • 中央总账:由单一网络、机构或个人控制的记录。(邪恶银行就是一个很好的例子)
  • 冷藏:将加密货币“离线”移动的过程
  • 密码资产:并非所有的密码都被用作货币手段，因此我们有这个术语。即CryptoKitties，或不可替代的可收集代币。
  • 密码学:编码和解码的科学或艺术。
  • 道:去中心化的自治组织。一个组织，其中程序化的规则集取代了传统公司中最集中的部分。
  • DApp:一个分散的应用程序。通常存储在一个区块链上，托管在许多分布式服务器上。
  • 去中心化:(通常是一个被行业营销人员过度使用的时髦词)不受一方控制的网络设计。移除第三方。(P2P)
  • DEX :去中心化交易所。
  • 分散:存放在多个地点。
  • 分布式/共享分类账:共享、可复制和同步数据的协议，分布在多个网络上。(P2P、去中心化、开源)
  • DTL :数字账本技术。像区块链这样的分布式分类账背后的技术类别。
  • 锻造:在区块链港的加工过程，矿工没有集体奖励。伪造者保留交易费作为奖励。
  • 分叉:当区块链永久分裂成两条独立的锁链时。

在福克斯上，马修·巴德莱本补充道；

分叉:任何特定区块链使用的协议的变化。类似于软件或应用程序更新。会导致要么；

A.)(无争议):所有参与者都采用新的数据“链”。

B.)硬分叉(有争议):参与者分裂，一些人采用新协议，而另一些人保留旧协议，导致“链分裂”。

即原始链和新链继续独立运行。

  • 燃气:交易费用
  • 创世纪区块:区块链中的第一个区块。
  • 减半:当方块奖励减半时，这意味着采矿将会增加50%的费用。
  • 硬分叉:更改为区块链协议，使之前无效的区块/交易有效。所有用户都需要升级。
  • 硬钱包:可以安全存储加密货币的设备。
  • 散列能力:给定一件硬件可以挖掘一枚硬币的比率。
  • Hashrate :一个方块被发现的速度和相关数学问题被解决的速度。
  • 闪电网络:在区块链上运行的支付协议，每秒能够处理数十亿笔交易。
  • 主网:主网a区块链还活着。
  • 主节点:有投票权的节点。
  • Mempool :未确认交易池，排队等待矿工确认/添加到区块链。有些特定于BTC，因为该地址池在不同的网络上可以有不同的名称。
  • MFA :多因素认证。通过向身份验证机制提供两个或多个证据来确认身份的方法。
  • 挖矿:运行软件破解密码谜题产生奖励。
  • 采矿设备:专门设计用于处理区块链矿石的计算机。多个高端图形处理器(GPU)来最大限度地提高处理能力。
  • 多重签名:多重签名。一种担保形式，需要一个以上的签名来批准交易。
  • 节点:一台拥有一个区块链网络副本的计算机，同时也产生确认以支持交易。
  • 开源软件:版权所有者授予用户研究、修改和发布内容的权利的计算机软件。(维基百科就是一个很好的例子)
  • 纸质钱包:印在纸上的加密货币钱包公/私钥。
  • 私钥:允许访问加密货币钱包的私人号码。
  • 公钥:从交易所/钱包发送或接收加密货币所需的一组数字/字母。
  • PoC :证明容量是一种共识机制算法，允许网络中的采矿设备使用其可用硬盘空间来决定采矿权。(谢谢， 塔姆桑切斯 为此通过推特补充！)
  • PoS: Proof-of-Stake ( 不，crypto不是一片sh**，虽然我妈会和你争论… )提出的未来共识算法。PoW要求证明者执行一定量的计算工作，PoS系统要求证明者显示赌注的所有权。是那些持有硬币的人做的。基本上，获得持有硬币的支付利息。
  • PoW :工作证明。那些拥有权力、时间和精力来获取回报的人进行的开采。制定为阻止拒绝服务攻击的措施。密码交易功率被称为“随机数”，或仅使用一次的数字。
  • 预开采:开发商在取链直播前为自己开采区块的行为。
  • 隐私币:纯匿名密码。允许私人信息和交易。
  • 协议:网络的一套规则。
  • 伪匿名:大多数区块链的分类账都是伪匿名的。每笔交易都是公开的，但是交易者的身份信息是保密的。
  • 二维码:可扫描的“条形码”。用于减少加密交易中的错误。
  • 中本聪:创造比特币的人创造的角色。这属于技术范畴，因为Satoshi很可能是来自未来的机器人。😜)
  • 分片 : ( 不是“分片”我应该清理我的自动更正，我应该是一位女士……)区块链中数据的水平分区。
  • 智能合约:存储有特定逻辑运算的不可更改的协议。
  • 快照区块:符合叉架要求的区块高度。这是一个模块，在此模块将拍摄分类账的快照，以确定分叉分类账的余额。
  • 软分叉:向后兼容的加密，这样货币就不会分裂成硬分叉。
  • 软钱包:存储加密货币，纯粹作为软件文件存在于计算机上。
  • 测试网:开发者和用户可以测试交易的测试网。
  • 令牌化:加密信息变成一串数据。然后可以发送和存储该字符串。在钱包之间发送并存储在区块链上的数据被令牌化。
  • 交易:交换&存储的数据添加到区块链。
  • 交易费:采矿的费用或报酬。
  • 钱包:存储密钥和创建加密货币交易的软件。
  • 钱包地址:接受/验证交易所需的两个加密密钥的公共部分。
