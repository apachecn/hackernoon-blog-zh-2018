区块链圈研讨会的经验教训

密码市场一天比一天血腥。但这是否意味着区块链技术失败了呢？

上周五我们开始了 工作坊 系列 区块链圈ConsenSys中 Rent24 。白天我们浏览了该技术最近的使用案例。以下是2018年20个引人注目的用例:

1。房地产

Brickblock 令牌化了区块链的第一个房地产。 Indiegogo 是 令牌化 房地产股票。

2。投票

150名西弗吉尼亚人通过一个区块链应用程序为 投票 。南韩 正在区块链测试 电子投票。日本城市筑波 完成了 区块链地方发展项目投票系统的测试。

3。身份解决方案

瑞士联邦铁路正在 测试输出 身份方案。安特卫普市是 测试jolo com为行政任务。 莫尼 是 为难民提供 一个区块链供电的id。

4。股票代币发行

NEUFUND在它的ETO(股票代币发行)平台上列出了第一批公司。最近，他们 通过ETO筹集了【300万美元。

5。海上保险

Insurwave ，一个区块链驱动的海上保险平台(由EY，Guardtime，A.P .穆勒-马士基，微软)现已投入 商业使用 。

6。加密银行

(与Solaris bank)推出 授权加密和菲亚特银行。

7。外汇交易&双边支付

德国商业银行和蒂森克虏伯 复制了 一笔使用区块链的外汇交易。高盛和摩根士丹利 使用了IBM解决方案 进行双边支付。

8。食品安全

澳洲国家运输保险检测区块链牛肉出口。沃尔玛 要求 选定的供应商使用区块链。美国乳品公司美国奶农(DFA) 是 与创业公司 ripe.io 合作 。

9。回购协议

俄罗斯主要银行 Sberbank 完成了 一份智能合约动力回购协议。

10。证券交易所

纽交所是 推出 一家比特币交易所，六家至 于2019年在区块链发售 证券， SAX 使用 区块链记录持股情况并管理清算。

11。令牌化所有权

Mattereum特殊化 一把价值900万美元的斯特拉迪瓦里小提琴。安迪·沃霍尔的画被符号化后在区块链出售。

12。许可&知识产权

埃森哲和区块链创业 数字资产 推出了 一款软件许可管理应用。

13。证书

新加坡 的公司可以在 SP(新加坡电力)集团的区块链上获得 可再生能源证书。

14。AdTech

丰田 与区块链新创企业合作，将广告欺诈降至最低。

15。患者数据

BlackBerry 正在网络上 工作 以支持患者数据的区块链解决方案。某医大医院在台 推出 一个以区块链为动力的病历平台。

16。土地注册处

澳大利亚新南威尔士州政府正在计划试验一种土地注册的区块链解决方案。非洲第一个多国土地登记处 即将在肯尼亚和加纳启动 。

17。电网&对等能源市场

韩国政府正在 为基于区块链的供电系统 工作。

18。点对点支付

日本电信公司软银 完成了 跨运营商P2P支付服务的概念验证。

19。自动化设备

沃尔玛 提交 一项新专利申请，在区块链进行自动化无人机送货。

20。数字证据

英国司法部正在 探索 使用区块链保护数字形式的证据。
