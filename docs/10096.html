计算BAC:

def calculateBAC(shotsConsumed, weight, genderConstant):
    newBAC = shotsConsumed/(weight*genderConstant)*50
    return newBAC

该函数将接受三个参数:

  1. 消耗的镜头
  2. 重量
  3. 性别不变。

该函数将计算新的血液酒精含量，并返回该值。
