**1。区块链标准化系统成为中国信息产业部的优先事项**

我们之前[报道过](/@BlockEx/13-03-2018-biggest-stories-in-the-cryptosphere-3060706e483e)中国工业和信息化部(IT)呼吁该国遵循区块链标准体系。该倡议现已成为[2018 年议程](https://www.coindesk.com/blockchain-standardization-tops-chinese-it-ministrys-2018-agenda/?utm_content=buffere1b73&utm_medium=social&utm_source=twitter.com&utm_campaign=buffer)上的优先事项。构成议程的七个领域中有四个与区块链技术的潜在标准化计划有关。首先是成立一个委员会，目标是建立一个标准化的区块链结构。其他举措包括促进与全球标准化组织的合作，以及研究智能合同和分布式账本框架。

**2。谷歌正在进行区块链项目，同时禁止区块链广告**

![](img/d44c6924ef47c4201e1a6877099f6069.png)

据[报道](https://www.bloomberg.com/news/articles/2018-03-21/google-is-said-to-work-on-its-own-blockchain-related-technology)称，谷歌目前正准备利用区块链技术来提高其竞争力并支持其云业务。这一决定在一定程度上是由于越来越多的初创公司将新兴技术用于业务，例如记录交易或处理数据。谷歌的 Alphabet Inc .子公司正在开发自己的分布式数字账本，然后将作为一项服务提供给其他公司。白标版本也将提供给公司使用。这家科技巨头此前还申请了一项“防篡改”审计系统的[专利](http://appft.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&u=%2Fnetahtml%2FPTO%2Fsearch-adv.html&r=1&p=1&f=G&l=50&d=PG01&S1=20180083786.PGNR.&OS=dn/20180083786&RS=DN/20180083786)。然而，本月早些时候，谷歌[决定禁止](https://cointelegraph.com/news/google-reveals-two-blockchain-projects-as-it-eyes-data-transparency)所有与加密相关的广告，区块链也在禁令之列。该公司尚未对这一特殊情况发表评论。

**3。九家马来西亚银行联手打造贸易融资区块链应用**

![](img/c1c6934e6353f94d18d2ac1141128110.png)

马来西亚中央银行(BNM)副行长 Jessica Chew Cheng Lian 宣布，九家马来西亚银行正在合作开发区块链贸易融资应用程序。行长在 2018 年亚洲银行家数字金融大会期间发布了这一信息。她讨论了银行业需要如何适应新兴技术带来的变化。活动期间讨论的一项倡议是促进银行之间的合作。此外，中央银行还成立了一个 API 实施小组，由来自金融和金融科技领域的成员组成。这是由于[开放应用程序接口](http://searchmicroservices.techtarget.com/definition/open-API)(API)的日益流行。目标是让该行业变得更具创新性和竞争力。

**4。比特币在荷兰法院被归类为“可转移价值”**

![](img/4dfec3f0cc220a28b3db88d77ca2337a.png)

根据 3 月 20 日公布的一份文件，比特币[已被荷兰一家法院宣布为“可转让价值”](https://cointelegraph.com/news/dutch-court-finds-bitcoin-a-legitimate-transferable-value)。一家非上市公司被法官勒令向原告支付 0.591 比特币，这笔钱被拖欠。选择是要么偿还债务，要么宣布破产。由于加密货币符合“财产权”的描述，原告有法律依据在法庭上提出异议。此外，法院预计该公司将以相同的货币偿还比特币欠款，而不是使用法定货币。这可以被认为是该国加密货币的一个进步，即使该国还不认为比特币是一种货币。
