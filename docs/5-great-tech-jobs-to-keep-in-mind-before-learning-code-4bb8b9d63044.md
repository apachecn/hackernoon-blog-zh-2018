如果你像我一样是一个代码初学者，在开始学习编程语言之前，先来看看技术领域的一些职业怎么样？

在寻找技术领域的工作时，最常用和最需要的语言有很多，但在开始学习代码的漫长道路之前，最好调查一下哪种技术职业与你的关系更大。下面的列表是开始搜索的好地方。你为什么不看看一些需求量很大的技术职业呢？

在加入一个职业之前，有几种方法可以更好地了解它。你可以询问该领域的专业人士，了解主要的工作任务是什么。你可以做一个工作搜索来检查所需的技能和平均工资。你可以在 LinkedIn 上关注一些公司，了解他们的文化以及对工作环境的期望。但最重要的是要明白，在科技领域的各种职业中，哪一个最符合你的需求。在你卷起袖子开始努力在就业市场中定位自己之前，你需要清楚这一点。
