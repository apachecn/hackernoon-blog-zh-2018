![](img/2ba297bccfc6bba3098cf1486d9bd63f.png)

## 请遵循以下步骤:

1.  [登录您的 Algolia 账户](https://algolia.com/)
2.  转到您的应用> API 密钥
3.  将您的 API 键(应用程序 ID 和管理 API 键)复制并粘贴到扩展中
4.  手动与您的 Cosmic JS Bucket 内容同步，或设置自动同步以使 webhooks 自动向 Algolia 添加记录。
