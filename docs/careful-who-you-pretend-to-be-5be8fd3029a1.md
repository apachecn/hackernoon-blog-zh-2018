# 小心你装的是谁！

> 原文：<https://medium.com/hackernoon/careful-who-you-pretend-to-be-5be8fd3029a1>

打开你的脸书或 Instagram。现在就做。你看到了什么？

关于我们如何在社交媒体上只描绘最好的自己，已经有很多报道了。当我们并不真正感到快乐时，我们会发布状态来显示我们是快乐的。我们放上我们最好的图片，这些图片已经过过滤和增强。我们展示我们的兴趣，我们的成就，我们的征服，包装它们，为最大的喜欢计时。

大多数文献都集中在这种行为的负面。是的，每个人都在社交媒体上假装(强调是一个更好的词，但假装更好地表达了观点)，我们只能看到我们所关注的人最好的一面，而现实并不完全如此。

这在我们的脑海中引发了我们自己脆弱和完整的真实自我(我们非常熟悉)与我们追随的那些增强和包装的自我(我们不太熟悉他们的现实)之间的比较，并导致了消极的感觉，感觉不值得，不快乐。

已经有研究证明了这一理论，我不对此进行辩论。我自己就是这种情况的受害者，我也知道其他人也是受害者，他们已经完全避开了社交媒体。

但是，和大多数故事一样，我认为这也有积极的一面。毕竟，在社交媒体出现之前，在他人面前假装就已经存在了。

我们假装比实际更熟练。我们假装比实际上更感兴趣。我们假装比实际上更关心和关心。以至于我们甚至为它创造了一个术语。假装直到你成功。

库尔特·冯内古特说过，“我们是我们假装的样子，所以我们必须小心我们假装的样子。”

我在将近十年前开始写这个博客，我发布的博文比我在 Instagram 上发布的照片还多。我一直认为这个博客的主题是“给自己的建议”。也就是说，我写事情的本来面目，也写事情的本来面目。只有在过去的几年里，我才变得越来越自信，敢于直言不讳地说出自己哪里做得好，哪里做得不好，哪里在学习。

正如库尔特·冯内古特所说，这个博客帮助我塑造了一个人，就像许多和我有过密切交往的人一样。

尽管人们在社交媒体上发布生活亮点的负面后果，同时保持低调，允许他们的追随者假设扭曲的现实，这种行为有助于发布者自我感觉良好，渴望成为他们展示自己的人，并最终实现这一目标。

上周六，一位朋友观看了我的单口喜剧表演，他分享了两张照片，并对我说:“[生活](https://hackernoon.com/tagged/life)大部分时间是第一张照片，因此需要第二张。”

![](img/b79b2c4ac1d0fddfe82f567bbe9f2152.png)![](img/2ce445c9fe4c11f9f2972c02820fc705.png)

或许所有在社交媒体上发帖的人也是如此。可能他们的生活大部分时候都是第一张图，他们贴出来的图结果让他们更开心。让他们休息一下。
