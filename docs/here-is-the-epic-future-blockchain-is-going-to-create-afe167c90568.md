# 开业

它实际上被称为分散自治组织，我用“开放业务”来使它听起来简单。开源社区是像 Linux，Google 等组织的支柱，他们很少或者没有政治。我说的不是组织，而是他们的项目。

比如 Hyperledger 就是 Linux 基金会的一个项目。它将帮助您构建私有区块链的应用程序。世界各地的人们都为这个项目做出了贡献。如果你做得好，你是谁或你是什么并不重要，你的代码将被合并到源代码中。你的工作得到了认可。

我观察到知道如何玩廉价政治的人胜过努力工作的人。在一个去中心化的自治组织中，每个人的工作都会被记录在一个透明的区块链上。你的工资将由评估你工作的系统提供。

开放业务将拥有开源项目的未来。任何地方任何有商业想法的人都可以创建一个分散的组织。任何人都可以成为它的一部分，程序员可以根据他的 git 档案自动加入系统。该组织的每个成员的报酬将完全基于每个人的贡献。

# 制片人的未来将会很美好

你可能已经对 ***【淘汰中间人】*** 这些词感到厌烦了。让我告诉你为什么这是一件大事。如果我必须在亚马逊上卖一本书，他们会抽取销售额的 50%作为佣金。写一本书是一项艰苦的工作。写一本书需要几个月的时间。

找出版商要花两倍的时间。这些出版商有他们自己的偏见( [*JK 罗琳被 12 家出版商*](http://riseupeight.org/jk-rowling-harry-potter-books/) 拒绝)，即使你的作品很好，也永远不会被发现。万一你运气好，你的书出版了。大部分收入归出版商所有。

如果你自己出版这本书，你必须努力推销它。最重要的是，看到亚马逊被大幅削减真的很痛苦。只有少数人敢于挑战这些挑战并成为制片人。几乎每个行业的生产者都受到剥削。

# 自由职业者呢？

Upwork 占我收入的 25%，如果我为一个项目挣 1000 美元，他们拿 250 美元。这是一个很大的数目！那么区块链有什么帮助呢？Upwork 有基础设施成本，他们需要支付服务器、数据库、维护等费用。在以太坊这样的公共区块链上，基础设施成本大幅降低。从以太坊读取数据是免费的。

在 Upwork 所基于的集中式基础设施上，人越多，需要更多的服务器来处理负载，也需要更多的资金。在一个去中心化的网络上，人越多，需要的钱越少，因为负载被用来支持区块链网络。

在基于区块链的平台上，费用可能低至 4%-2%。我只需支付大约 40 或 20 美元作为佣金。与 250 美元相比，这要低得多。 [Vanywhere](http://vanywhere.com) 正在建立一个基于区块链的平台。
