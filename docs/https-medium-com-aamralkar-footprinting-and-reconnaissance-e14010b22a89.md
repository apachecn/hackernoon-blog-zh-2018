# 足迹和侦察

> 原文：<https://medium.com/hackernoon/https-medium-com-aamralkar-footprinting-and-reconnaissance-e14010b22a89>

## 足迹到底是什么？

![](img/fa5381d35406feb1d1f56b8517d428aa.png)

> 用通俗而简单的语言来说，“足迹”在安全术语中是收集尽可能多的目标组织信息的过程

信息包括:-

物理位置、域、员工数量、URL、VPN、电话号码、IP 地址等。

足迹威胁？

1.  社会工程:最简单的，不需要任何工具就可以完成。
2.  经营兴趣减少
3.  企业间谍活动
4.  信息泄露
5.  系统和网络攻击

## **足迹目标**

1.  收集网络信息
2.  收集系统信息
3.  收集组织信息

**足迹不同的方法。**

1.  通过**社交媒体**的足迹，这是最容易做到的。大多数攻击者会创建假帐户/id，并试图收集尽可能多的关于目标组织的信息。
2.  **足迹**通过**搜索引擎**像必应、谷歌和 duckduckgo。我最喜欢的是 duckduckgo。攻击者还会寻找缓存和归档。一些好的工具有 netcraft，shodan，pipl，Google Earth。以便进行足迹分析。
3.  在工作现场留下足迹。黑客将会知道组织正在研究什么工具和技术。
4.  通过像谷歌警报，推特警报，雅虎警报的目标监测。
5.  另一个好方法是通过谷歌入侵数据库和高级搜索查询。查询字符串可以用于搜索，也可以用作关键字。也可以利用谷歌高级搜索运营商。例如“标题索引”列出所有打开索引的站点。securityfocus.com、hackersforcharity.org/ghdb 是少数几个你可以获得大部分信息的网站。
6.  网站足迹正在监控目标组织网站。Web 服务器详细信息、目录结构、开发者电子邮件 id 是一些常见的信息。还有工具，我们可以镜像整个网站。回溯的网站信息可以从 archive.org 提取。
7.  电子邮件跟踪用于跟踪电子邮件。电子邮件是用来收集信息，以执行社会工程和许多其他攻击，垃圾邮件。
8.  DNS 信息攻击者可以获取网络中的主机。黑客可以获得 A，CNAME，PTR，MX，NS，HINFO 的记录。有很多命令行工具可以用来获取 DNS 信息。nslookup 和 dig 是这些工具中最常见的。
9.  whois 攻击者执行 WHOis 以了解特定域背后的 WHOis？ARIN，非洲，成熟。APNIC、LATNIC 是 RIR(区域互联网注册中心)。我们可以从 WHOis 获取信息，如电子邮件、域名所有者、地址、域名服务器、注册商。
10.  网络足迹
11.  通过社会工程的足迹。偷听，肩膀冲浪，翻垃圾箱。
