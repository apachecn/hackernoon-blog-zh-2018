# 连接到您的铲斗

1.  [转到 Cosmic JS](https://cosmicjs.com/) ，通过转到你的桶>设置>导入/导出，导入这个 repo 中包含的`bucket.json`文件。
2.  进入`index.vr.js`文件，找到配置部分并添加您的 Bucket slug。
3.  启动连接到您的桶的应用程序:
4.  在[http://localhost:8081/VR](http://localhost:8081/vr)的浏览器中玩 VR 体验
