# 如何使用

*   为您各自的浏览器安装 Metamask 扩展并创建一个帐户。
*   开始上传文件。

# 特征

*   使用元掩码创建帐户，因此它允许比传统云平台更好的匿名性。
*   从本地存储上传文件。
*   无限存储(目前为*)。*
*   *免费存储。*
*   *通过电子邮件和公钥共享文件。*

*![](img/e52d4eaa43aa1b8c01fb12ce730c3b58.png)*

*Share documents using Email/UserId/PublicKey*
