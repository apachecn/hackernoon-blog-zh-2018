# 第二部分:认知过度刺激、抑郁和多动症

> 原文：<https://medium.com/hackernoon/part-2-cognitive-overstimulation-depression-and-adhd-e404e3a5e729>

![](img/8b21d8f61051c151d9583ce137c92423.png)

在我的上一篇帖子中，我写了关于社交媒体、抑郁症和多动症的内容。在这第二篇文章中，我将写关于精神过度刺激、抑郁和多动症。

人们认为第一台电视过度刺激大脑到了有害的程度。我们说的是没有音频的黑白电视，远没有今天的电视刺激。任何影响尚不清楚，但马歇尔·麦克卢汉推测电视会使大脑对刺激变得不敏感。*甚至只是第一，基本的电视*。看电视的时候，成千上万的电点会移动和变化。大脑不能处理所有新奇的视觉信息，只能处理轮廓:你在屏幕上看到的东西。试图吸收所有不自然的新鲜感，大脑受到过度刺激。他认为精神在那之后变得麻木，保护了自己。

我们总是被新奇的视觉信息所刺激。我们这样进化是为了更好地注意食物(*移动*猎物)和危险的威胁(*在灌木丛中沙沙作响*)。电视、电影和视频游戏不断地传递新奇的视觉信息。它在精神上迷人和令人兴奋，这提供了短期价值，但可能会用刺激和兴奋轰炸头脑。成年人可能不会注意到，但年轻人可能会。

不断被这种新奇的视觉信息过度刺激的年轻人可能不太容易接受课堂上的新信息。如今，随着高清屏幕——高密度的新颖视觉信息——大量接触电视节目、电影和视频游戏可能会使这种影响更强。

年轻人可能会过度接触信息和人为的快乐，而不太接受真实世界的信息，无论是在社会上还是在学术上。

除非独处是为了重新训练大脑的接受能力，否则人们可能不具备完成工作或学校的基本需求所需的高效精神状态。

环顾大学图书馆或体育馆，很少有人在没有额外刺激、没有耳机或不断检查手机的情况下专注于某件事情。许多人上瘾了，对刺激不敏感，需要一些人工刺激来继续工作。他们没有生活中的自然刺激和意义——因为他们脱敏了——而是依赖人工刺激。我们可能并不总是专注于我们的学业或专业工作，但脱敏可能会使问题变得更糟。
