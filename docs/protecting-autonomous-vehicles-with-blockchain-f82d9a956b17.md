# 用区块链保护自动驾驶汽车

> 原文：<https://medium.com/hackernoon/protecting-autonomous-vehicles-with-blockchain-f82d9a956b17>

![](img/76f196970756e8aac54b653080857e12.png)

“网络事故是世界上每个汽车制造商的问题。这是公共安全问题。”—嫁给通用汽车首席执行官巴拉。

自动驾驶汽车的未来有望极大地改善我们的生活。想象一下不再有压力的通勤和严重事故的大量减少。随着各种技术和汽车公司竞相将自动驾驶汽车带到我们的道路上，这些和其他在便利性和安全性方面的重大进步可能比许多人意识到的要近得多。

然而，就像所有连接到互联网的东西一样，利用安全漏洞可能会导致灾难性的后果。研究人员发现了一些重大缺陷，这些缺陷使他们能够接管联网汽车的控制权，例如臭名昭著的吉普切诺基事件导致 140 万辆汽车被召回，并增强了确保联网汽车和自动驾驶汽车绝对安全以避免黑客入侵的重要性。

在考虑全面可行的安全解决方案时，区块链可能会被证明是理想的，因为它是防篡改的。区块链的解决方案将所有这些信息整合到一个安全的系统中，以确保准确性和透明度，同时大大降低黑客入侵的可能性，而不是像目前这样依赖大量独立的孤岛来存储和管理数据。

虽然预测的数字有所不同，但大多数专家认为，在未来几年内，大量的自动驾驶汽车将出现在我们的道路上。随着这一数字的增长，数据量也将大幅增加。虽然这些数据在许多方面都非常有用，但它也为恶意利用车辆和存储的相关信息提供了机会。当然，攻击无人驾驶汽车的后果可能是灾难性的。因此，必须确保所有网络安全政策和工具都经过精心设计和部署，以保证安全。

随着联网汽车的扩张和自动驾驶汽车的快速到来，保持现状并不是一个合理的选择。该行业将需要改变其做法，加入到一个单一的系统中，以确保所有车辆和道路数据的安全性和准确性。只要该系统开发得当，并采取全面的安全措施，我们都可以期待一个包括自动驾驶汽车的未来，从而极大地提高我们日常生活中交通需求的安全性和便利性。
