# 安全令牌和清算优先权

> 原文：<https://medium.com/hackernoon/security-tokens-and-liquidation-preferences-78105d51905f>

![](img/975a91abb744578fbb987824f823d792.png)

*2018 年 8 月 8 日
加州三藩市*

在任何一门金融 101 课程中，学生都会学到，当一家公司破产时，那些向该公司提供贷款的人将先于那些持有该公司股权的人得到偿付。创业公司的本质(无论他们是否筹集了股权或象征性资金)是失败率必然很高。那么，为什么公用事业代币购买者目前不寻求保护自己免受清算事件的负面风险，在此期间，他们将有权收回他们承诺的 0%的资金？

输入安全令牌持有者。因为它们是证券，证券代币将赋予持有者某些清算权利，这些权利实际上是有价值的，特别是因为许多目前通过代币发行筹集资金的公司是未杠杆化的，它们的资产负债表上几乎没有债务。事实上，除了在区块链上以数字形式表示传统资产之外，安全令牌的格式使发行者比以前更具创造性。现在，公司可以发行代表收入或利润分享安排权利的代币，这意味着证券代币持有人可以先于股权或债务持有人获得公司创造的价值。

说了这么多，重点是什么？公用事业代币购买者会在意清算偏好吗？在与该领域的机构参与者交谈后，一些人认为，如果初创公司只是为可以在公司平台内使用的其他类型的公用事业令牌筹集后续资金，那么首先就可以避免资金耗尽的可能性。不过，在这一点上，该公司是否能够吸引公众，在一个拥挤的生态系统中给他们“第二次机会”，还有待分晓。在这个生态系统中，大多数参与者似乎都患有“发光物体”综合症，他们更被迫出售自己的代币头寸，而不是成为一个忠实的支持者，等待风暴结束。
