# 智能环境创造更安全的世界

> 原文：<https://medium.com/hackernoon/smart-environments-create-a-safer-world-a4040bb360a>

![](img/02c01461f393f4ffb15911818f37cfc2.png)

*“智能城市正在取得令人难以置信的进步——尤其是在欧洲——利用人工智能、传感器、数据分析、连接等成分技术来改善我们的生活。智能城市使我们社区的日常生活更加高效，减少对环境的影响，改善城市服务，创造就业机会，并有益于经济。它们让我们更安全，更容易接近。到 2050 年，全球将有三分之二的人口居住在城市，现在是利用技术潜力改善城市生活的时候了。”—消费者技术协会市场研究高级总监 Steve Koenig。*

随着物联网(IoT)的激增，我们可以期待一个更安全、更宜居的世界。我们的城市变得越来越拥挤，这给城市居民带来了许多危险和不愉快的现实。根据联合国的数据，目前世界上 55%的人口居住在城市，到 2050 年将飙升至 66%。这种城市化加上总人口的增长，预计在未来 30 年左右的时间里，城市人口将增加 25 亿。

显然，如果不采取激进的行动，这种情况是不会改善的。幸运的是，由物联网驱动的智能环境有助于缓解这些挑战。正在开发新的服务，以允许对威胁做出更快的响应，例如远程连接的闭路电视和自动事件检测。IOT 领导的人群管理解决方案将增加对生活在繁忙地区的城市居民的保护。智能街道照明是另一项旨在降低犯罪率和改善驾驶条件的措施，这两项措施都将提高城市居民的安全。

尽管对自动驾驶汽车存在许多担忧，但现实是这些机器人汽车最终将大大减少道路事故。绝大多数碰撞是由人为错误或注意力分散造成的，我们城市的拥堵正使通勤变得越来越困难。自动驾驶汽车有可能通过更有效的驾驶从根本上减少拥挤，同时大大减少道路上的轻微和严重事故。

物联网正在提升所有城市服务。从公共资源(包括医疗、交通和教育)效率的提高，到更好的城市管理带来的显著经济增长，智能环境为我们所有人带来了一个更安全、更可持续的世界。
