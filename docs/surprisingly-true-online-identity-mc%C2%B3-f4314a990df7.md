# 这有什么区别吗？

互联网创建于 1973 年，它基本上是一个无需许可的设备网络，旨在存储和交换数据。然而，这些设备从未在全球范围内与人或他们的身份有任何物理绑定。任何人都可以选择一个名字，做他们想做的任何事情，而无需承担任何责任或易于追踪。
