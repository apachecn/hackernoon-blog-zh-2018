# 构建安全令牌项目社区的重要性

> 原文：<https://medium.com/hackernoon/the-importance-of-building-a-community-for-security-tokens-8f1a71a959a6>

![](img/4a0308b389de5df070e0ebf3b514f5af.png)

*加州三藩市【2018 年 11 月*

作为[旧金山安全令牌会议](https://www.meetup.com/San-Francisco-Security-Token-Meetup/)和[纽约安全令牌会议](https://www.meetup.com/NYC-Security-Token-Meetup/)的组织者，我一直努力促进与我直接相关之外的讨论，以提高对现实世界资产令牌化、托管这些资产、合规交易这些资产等各种障碍的认识。

当我第一次创建这些 Meetup 群时，我没有想到我们会得到这么多的关注。我错了。我们围绕安全令牌组织的第一次 SF 聚会有超过 200 人回复，第二次 SF 聚会也是如此。人们渴望更多地了解这个领域，这一点从人们挤进 Kearny 街 1011 号的小“节点”聚集地听 Mike Belshe、Josh Stein、Jay Thakrar、Vijay Chetty 和我谈论安全令牌就可以看出。一些观众站在楼梯间，只是为了偷听谈话。

我通过组织这些会议了解到，人们渴望了解美国的法规将如何影响加密和区块链行业，以及所有 ICO 的处理方式是否会慢慢变得更像 STO。简而言之——STO 的 crypto 的拯救之恩将防止 2017 年和 2018 年通过 ICO 首次亮相的许多这些替代硬币的痛苦市场崩溃吗？这个存在主义的问题似乎贴近许多与会者的内心——他们中的一些人离开了谷歌、优步、脸书等科技公司的高薪工作，以追求成为一名密码企业家的梦想。

我可以肯定的是，STO 并不是每个产品或公司的完美用例。其实 ICO 的也不是。然而，许多人将 STO 视为救星的事实有助于将许多这些新颖的想法引入安全令牌主流，以帮助确保高质量的 STO 能够筹集资金，以达到他们的融资目标，推出高质量的产品，并最终让他们的 STO 进入高质量的安全令牌交易平台。

即使 ICO 被视为证券，我们在这些 Meetup 小组中汇集的 Sharespost 和其他工具也为令牌化证券在交易平台上合规交易和伴随的流动性铺平了道路，因此对非流动性令牌化资产的担忧将很快成为过去。

这些聚会已经成为社区就如何继续构建安全令牌环境交换意见的机会。他们还教育更多的社区成员了解安全令牌所面临的障碍。随着社区的不断发展和分享想法，我们可以找到促进安全令牌发展的方法。对于我们应该如何继续促进社区中围绕安全令牌的讨论，您有什么想法？请在下面的评论中告诉我。
