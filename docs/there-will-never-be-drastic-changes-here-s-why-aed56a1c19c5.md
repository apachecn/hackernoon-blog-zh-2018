# 永远不会有“剧烈”的变化，原因如下。

自从人类诞生以来，科技一直在稳步发展，不管你多么想相信生活在 21 世纪的“我们”是科技“变革”的缩影，但我们不是。你被以自我为中心的偏见所愚弄，我们相信人类有史以来目睹的最大变化就发生在我们身上，我们相信，如果是我们的前几代人，他们不会应对。我经历过，从使用个人电脑到高性能便携式笔记本电脑，从 Orkut 到脸书到 Snapchat，从座机到诺基亚到超级优化的 iPhone。

事情在变化，我们认为接受它是我们的责任。我们现在生活在互联网上，想象我们在 18 世纪拥有的东西和技术会让你确信你是一个妄想型精神病患者。但是我们已经超越了，在被这些“变化”逗乐的同时，我们也住了下来，接受它们本来的样子。你不需要在一天甚至一年内接受它们，重要的是你为了自己的利益和舒适而接受它们。

我们有 76 亿人陪伴着我们在这个星球上，没有希望在不久的将来穿越时空，我们肯定会死在这个星球上。在人类的一生中，我们没有一次能够战胜自然的基本法则，我们受到身体的物理、化学和生物限制的束缚。

我们预见的最剧烈的变化是自动化，而且，一旦看到人口增长的下降，我们可以相信这将是一件好事。机器学习带来的变化将帮助我们通过计算在更大的范围内设定事物。我们大多受到能源和时间的限制，人工智能可以为我们改变这种情况，更好地管理我们的计算，进一步推动我们的能力，并在自然法则的指引下跨越宇宙。
