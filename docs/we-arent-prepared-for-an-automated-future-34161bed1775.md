# 什么是自动化？

我们生活在一个越来越受算法、软件、传感器、连接和数据管理和控制的世界。这些技术发展正在推动一场深刻的经济、社会和文化变革。

当前的大部分讨论都是关于工作自动化的。

人工和知识工作将越来越多地由智能和互联的机器来完成。

标准化程序将实现自动化。当然，我们还没有到那一步。例如，无人驾驶汽车和无人机送货系统还有一段路要走。但如果你跟踪这些领域的技术发展，很明显这种变化正在到来。

但是自动化革命不仅仅是机器人执行标准化的任务。

比如随着*优步*和 *Airbnb* 等公司的出现，我们正在见证**的协调自动化**。

# 自动化世界的新“技能组合”

那么，为了迎接自动化的未来，我们还应该做些什么呢？这里有三个技巧可以帮助每个人更好地为这个新世界做准备。

## 1 —“接口”

新技术使得更直接的“点对点”交易成为可能。这种交易越来越自动化。

但这并不意味着中介会完全消失。

相反，中介的“工作”将会发生转变。中介机构需要专注于促进这些更直接的交易形式。这意味着专注于用户界面设计。

> **未来的媒介将是“界面设计师”,在人类用户和互联的智能机器之间的新空间中工作。**

以银行业为例。

过去，银行“代表”他们的客户，并代表他们的客户进行交易，而今天(以及未来)的金融交易正越来越多地在双方之间进行，使用的是由“中间人”提供的联网机器和用户界面。

在新兴的智能合约中也可以看到类似的动态。在自动化的未来，律师的角色将不再是为客户谈判和起草传统协议，而是促进双方之间的直接交易。这里应该注意的是，不同的用户(基于不同数据集的不同期望)将需要设计不同的用户界面。

此外，在“更智能”机器的新世界中，更需要便于人类用户有效操作机器的用户友好界面。中介“增加价值”不是通过他们在交易中代表客户的能力，而是通过提供和持续测试使用户体验尽可能平滑的界面。

在“接口”的新世界中，设计和测试技能变得至关重要。

## 2 —“共同创造”

有人可能会说，设计用户界面是一项主要落在技术专家身上的任务。当然，技术专家通常对互联技术的可能应用有更好的理解。

当然，实现一个特定的设计解决方案在很大程度上是一个需要编码技能的软件问题。

尽管如此，非技术专家仍然扮演着重要的角色。当“做得正确”时，技术专家和非技术专家的合作是不可思议的。

正如我在之前的一篇文章中所解释的，我们需要更多的技术专家和非技术专家的“奇怪组合”来共同创造向市场提供应用程序，为更广泛的受众提供有意义的体验。这是为企业和消费者增加价值的最有效方式。

科技和商业这两个领域之间这种创造性的伙伴关系有很多例子，它们对科技公司的成功至关重要。

自动化革命意味着每个人都需要接受这种合作方式。

为了让这种“零工经济”发挥作用，需要更加强调“剖析”(在工人*和*组织这两方面)。

请记住，声誉和信任将是自动化的(不太强调传统的中介和品牌声誉)。这意味着，为了让员工和组织在对等关系中具有吸引力，他们必须建立一个强大的档案，这是独特的和个人的。

“个人资料”不同于“品牌”，因为它既是一种形象，也是一种声誉记录。

关于声誉记录，我指的是公开披露的个人以往成就的历史，这种历史接受不断的审查，并由用户的反馈回路(积极的和消极的)来评估。
