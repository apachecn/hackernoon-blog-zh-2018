**可选触控 ID 验证**

我们还增加了可选的触控 ID，以提高应用内的安全性。当发送第一笔交易时，用户可以选择要求触控 ID 访问交易流。这将增加应用程序的安全性，甚至可以防止被盗、未锁定的手机被偷。Josep & Rosello 正在开发一个介绍性的工作流程，让用户选择他们是否希望使用触控 ID——他们设计的自定义弹出窗口应该会告知用户保护他们钱包的额外安全好处。
