# 比特币、加密货币以及所有怪异的行话是什么？

我知道要想出第一步是很困难的，但是也没那么复杂。一旦你知道了一些基本术语，你就能理解剩下的了。我的目的是排除任何多余的细节，直到你熟悉了基础知识。之后，你就可以自己做研究了。

你会看到诸如硬币、代币、密码和替代硬币等术语被使用，但为了简单明了，我将在本书的剩余部分使用密码作为指代它们的术语。

它们都是指分散的虚拟货币，您可以将其用于各种目的，其中一个目的是将货币价值转移给另一个人或组织。

然而，传统货币被称为法定货币。

法定货币是一种没有内在价值的货币(就像加密货币一样，不同于黄金或白银)，由政府法规或法律确立为货币。该术语源自拉丁语 fiat(“让它成为”，“它将成为”)，用于命令或法令的意义上。

法定货币在 20 世纪开始占据主导地位。自 1971 年理查德尼克松(Richard Nixon)宣布美元与黄金脱钩以来，一种国家法定货币体系已在全球范围内使用，各国货币之间的汇率自由浮动。

# 密码的用途是什么，它们是如何产生的？

虚拟货币的第一次尝试发生在 20 世纪 80 年代末的荷兰。

午夜时分，偏远地区的加油站遭到现金袭击，经营者不愿意让那里的警卫处于危险之中。但是加油站必须通宵营业，这样卡车才能加油。

有人想出了一个好主意，把钱放在当时正在试验的新式智能卡上，于是电子现金诞生了。卡车司机得到了这些卡，而不是现金，车站现在更安全了，不会被抢劫。

与此同时，占主导地位的零售商 Albert Heijn(一家大型杂货连锁店)正在推动银行发明一些方法，允许购物者直接从他们的银行账户付款，这最终被称为 POS 或销售点。

90 年代中期，PayPal 应运而生，它是最早允许人们直接在线汇款的系统之一。最终，它作为一种在线支付商品的安全方式而闻名于世，不用担心你的信用卡信息被网站窃取。

自 90 年代以来，人们一直试图创造数字货币，但第一种真正的加密货币是比特币，由一个名叫中本聪的神秘人或团体于 2009 年创造。

这种货币在其使用者之间直接起作用，不需要任何中介，如金融机构。

比特币的故事与操作系统 Linux 惊人地相似，它是由 Linus Torvalds(当时 22 岁)在上世纪 90 年代在他的宿舍里创建的，作为他的论文项目，他在网上免费发布了这个项目，供每个人随意复制和修改。

现在你最有可能在 Linux 驱动的设备上阅读这本书，或者如果你在网上购买了平装本，你的订单已经被 Linux 驱动的计算机接收和处理。

类似地，比特币已经被创造出来，并在网上免费公开发行，让所有人受益。这种技术有真正的手段让世界变得更好。

为了更好地理解密码在全球引发好奇心的一些原因，接下来我想分享一些故事。

# 与传统货币相比，加密货币是如何工作的？

所有的密码都建立在一个叫做区块链的技术基础上，它本质上是运行在世界各地成千上万台机器上的计算机代码，这些机器通过互联网组成了一个网络。

你可以把区块链想象成一个巨大的 Excel 表格，里面包含了曾经进行的所有交易。一旦交易被保存并被整个网络确认，它就不能以任何方式被改变。

所有这些都建立在现代密码学(因此得名加密货币或密码)的基础上，这是一门位于数学、计算机科学和电子工程交叉点的科学学科。

随着计算机的出现，密码术本身在第二次世界大战中发挥了关键作用，并且如今在我们的日常生活中无孔不入，即使我们根本不怎么考虑它。

也许区块链网络最重要的一面是，没有一个单一的组织、个人或地点能让你找到它。

相比之下，你的银行有一个办公室和一个你可以拨打的电话号码，他们管理的自动取款机，以及一个管理账户和交易的 IT 系统。他们的 IT 系统由他们自己的团队管理，并在称为数据中心的建筑物中运行，数据中心是专门为此目的而设计的。

在比特币的早期，它被更广泛地用于隐藏网络，以避免审查。人们可以在诸如“丝绸之路”之类的纯暗网网站上购买毒品之类的东西，但当局已经取缔了这些服务，并逮捕了涉案人员。

## 尽管如此，如果没有这些早期采用者，无论他们是谁，比特币可能都不会存在。

还有其他硬币提供更高水平的在线匿名，但比特币不一定是其中之一。如果有动机的话，当局确实有办法追踪交易和钱包背后的人。

虽然它没有银行账户或信用卡那么简单，但也并非不可能。

> 这位国土安全部官员说:“通过执法部门追踪那些(罪犯)并让交易所更加负责，我们的情况有了很大改善。”。“我认为(比特币)比人们认为的要合法得多。"

另一位接受美国消费者新闻与商业频道采访的网络安全专家坚持认为，比特币匿名的假象只是……一种假象。

“比特币基本上引入了一种我们可以绕过钱骡的情况，”RedSocks Security 的网络犯罪专家里奇·吉弗斯(Rickey Gevers)说，该公司检测并打击恶意软件。

但是，格弗斯说，“一开始(比特币)看起来很匿名，最后看起来也不怎么匿名。”
